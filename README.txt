vimからRen'Pyを開くプラグインです。

設定:
let g:renpyopen_use_vimproc=1 "1 ならvimprocを使います
let g:renpy_exepath="e:/Soft/renpy-6.14.1-sdk/renpy.exe" "renpy.exeのpathです

使えるコマンド:
RenPyExe ゲームを起動します
RenPyExeCurrentLine 現在行でゲームを起動します

This plugin allow you to execute a renpy game by vim.

Setting:
let g:renpyopen_use_vimproc=1 "If 1, use vimproc
let g:renpy_exepath="e:/Soft/renpy-6.14.1-sdk/renpy.exe" "the path of renpy.exe

Command:
RenPyExe
RenPyExeCurrentLine
